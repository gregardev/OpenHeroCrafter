# Open Hero Crafter

A free and open source character creation tool for Mutans and Masterminds 3rd edition. It works on the browser and it uses local storage. It is build on React + Vite with Typescript. 
---
Clone the repo and install with 
```npm i```
---
Run it in Development mode iwht 
```npm run dev```
---
Build it with
```npm run build```
